Шаблон итератор

Цель получение последовательного доступа к овсем элементам составного объекта.

Для чего используется:

Для получения последовательного доступа к овсем эелментам составного объекта, скрывая его внутреннее представление.

Пример использования:

- различные виды обхода составного объекта
- упращенный доступ к составному объекту
