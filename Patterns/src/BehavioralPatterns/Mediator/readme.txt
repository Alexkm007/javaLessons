Шаблон посредник

Цель:

Инкапсуляция способа взаимодействия множества объектов

Для чего используется:

Для опредления объекта, который инкапсулирует способ взаимдоействия множества объектов
и обеспечения слабой связи между этим объектами.

Пример использования:

- связи между объектами сложны и четко определены;
- нельзя повторно использовать объект, так как он обменивается информацией с другими объектами;
- поведение, распределенное между несколькими классами должно легко настраиватся без создания подклассов.
