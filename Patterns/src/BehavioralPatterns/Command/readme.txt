Шабблон комманда

Цель:
Инкапсулирование запроса в объект

Для чего испльзуется:

Чтобы задать параметры клиентов для обработки предленных запросов, создание очереди из этих запросов
или их колнтроля и поддержки отмены операций.

Пример использования:

- параметризация объектов выполняемым действием;
- опредлить запрос, ставить его в очередь или выполнять его в разное время
