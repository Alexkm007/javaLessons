Шаблон Strategy

Цель:
Взаимосвязь семейства классов.

Для чего используется:

Определение семейства классов, инкапсулирование каждого из них и организация взимозаменяемости

Пример использования:

- есть несколько родственных классов;
- необходимо иметь несколько вариантов поведения;
- в классе есть данные, о которых не должен знать клиент;
- с помощью условных операторов в клссе определено большое количество поведений.
